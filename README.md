# 🐉 Projeto HOTD

Aplicação inspirada no universo de **House of the Dragon**, desenvolvida com foco em prática de desenvolvimento web, consumo de APIs e organização de interface.

---

## 📌 Sobre o projeto

O **Projeto HOTD** é uma aplicação que tem como objetivo apresentar informações relacionadas ao universo de *House of the Dragon*, podendo incluir personagens, casas, episódios ou conteúdos dinâmicos.

Este projeto foi desenvolvido com fins de estudo e aprimoramento em:

- Desenvolvimento Front-end / Full Stack
- Estruturação de projetos

---

## 🚀 Funcionalidades

- 🔍 Listagem de personagens / conteúdos
- 📖 Exibição de detalhes
- 🎨 Interface amigável
- 📱 Design responsivo

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
